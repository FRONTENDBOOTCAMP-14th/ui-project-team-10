# 공통 컴포넌트 및 유틸리티

이 디렉토리는 애플리케이션 전반에서 재사용되는 공통 컴포넌트, 유틸리티 함수, 상수, 타입 정의 등을 포함합니다.

## 디렉토리 구조

```
src/common/
├── modern-reset.css        # 브라우저 기본 스타일 초기화
├── theme.css               # 테마 변수 (색상, 간격, 타이포그래피 등)
├── a11y.css                # 접근성 관련 유틸리티
├── base.css                # 기본 요소 스타일링
├── animation.css           # 애니메이션 및 트랜지션
├── layout.css              # 레이아웃 관련 유틸리티
│   ├── grid                # 그리드 시스템
│   ├── flex                # 플렉스박스 유틸리티
│   └── container           # 컨테이너 스타일
│
├── utils.css               # 유틸리티 클래스
│   ├── spacing             # 마진, 패딩 유틸리티
│   ├── typography          # 텍스트 관련 유틸리티
│   ├── display             # 표시/숨김 관련 유틸리티
│   └── position            # 위치 관련 유틸리티
│
└── index.css               # 모든 스타일을 임포트하는 메인 파일
```
