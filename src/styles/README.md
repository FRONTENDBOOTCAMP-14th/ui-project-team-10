# 컴포넌트 스타일

이 디렉토리는 애플리케이션의 컴포넌트별 스타일을 관리합니다.
전역 스타일 및 유틸리티는 `src/common/` 디렉토리에서 관리됩니다.

## 디렉토리 구조

```
src/styles/
├── Button/
│   ├── index-module.css
│   └── variants-module.css
├── Card/
│   └── index-module.css
├── Header/
│   └── index-module.css
└── ...
style.css              # variants-module을 모두 임포트하는 모듈
```
